现在的纯原生的 APP 很少见了，几乎都是原生中嵌者网页的 hybird APP，或者直接使用 React Native 开发。原生 App 中嵌者网页原生与网页中 JavaScript 的交互几乎无法避免，毕竟在自己的应用中嵌入网页不只是想让应用当个浏览器用。
Android 原生中加载网页的控件是 WebView，因此要想原生与 JavaScript 交互也一样必须通过 WebView 来实现。
